About Project

The project is the Facial Expression based Music player system made in Django with machine learning libraries like OpenCV and TensorFlow. Here the user has to detect their facial expressions through webcam and the songs recommended list will be displayed. Also user can make his own custom songs playlist for his convenience. This web app is highly beneficial as all things are done automatically using machine learning concepts.
